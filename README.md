# fgScorecard backend

### Initial Setup

```

// install pipenv, packages
$ pip3 install pipenv
$ pipenv
// run virtual environment
$ pipenv shell
// run migrations
$ python3 manage.py migrate
// start server
$ python3 manage.py runserver

```
